- **正宗哥吉拉**（100%~60%）— 深灰黑軀體、藍色原子吐息

## ✨ 主要特色

- **🌤️ 六關獨特場景** — 草原平野／中原大地／江南翠嶺／北方荒原／關西高原／蒼茫邊塞，各有不同地形起伏與地面配色（全白天晴空）
- **🚪 城門傳送門系統** — 斬殺 BOSS 後城門開啟發光傳送門，走進去過關
- **⚔️ 三種小兵** — 近戰兵／遠程弓兵（放冷箭）／強力騎兵
- **🧱 Minecraft 建造模式** — 按 B 切換，可放置／破壞方塊
- **🌐 中英雙語** — 開始畫面可切換 自動／中文／EN，自動偵測瀏覽器語言
- **🎯 三種難易度** — 新手村／高手／地獄
- **⏭️ 跳關功能** — 可選任意關卡開始
- **🏆 本地排行榜** — 依「闖關速度（高）× 擊殺數（中）× 剩餘武將（低）」綜合計分，localStorage 持久保存
- **📋 版本選單** — 可回玩 v1.0 ~ v1.5 任一歷史版本
- **🎆 通關絢麗煙火特效**
- **📱 iOS/iPadOS Safari 優化** — 觸控搖桿、visualViewport 處理、自動關陰影提升幀率
- **🎵 Web Audio 即時合成音效** — 零外部檔案

## 📐 技術細節

| 項目 | 說明 |
|------|------|
| 引擎 | [Three.js r160](https://threejs.org/) via CDN import map |
| 渲染 | WebGL，InstancedMesh 地形（單 draw call）|
| 音效 | Web Audio API 即時合成，零外部檔案 |
| 儲存 | localStorage（排行榜 + 語言設定）|
| 封裝 | 單一 HTML 檔案，約 6,400 行，無依賴 |
| 國際化 | 內建 zh/en 字典 + `data-i18n` 動態切換 |

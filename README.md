## ✨ 主要特色

- **🗺️ 全關場景變化** — 六關各有獨特天色（晴空草原 → 黃昏戰場 → 翠谷密林 → 赤焰沙場 → 幽暗夜戰 → 核爆紅天）
- **🚪 真實城門系統** — 斬殺 BOSS 後，城門開啟發光傳送門，玩家親自走進去過關
- **⚔️ 三種小兵** — 近戰兵 / 遠程弓兵（自動保持距離、放冷箭）/ 強力騎兵（騎馬、背旗、速度快傷害高）
- **🧱 Minecraft 建造模式** — 按 B 切換，可放置/破壞方塊
- **🐉 友軍動物** — 熊貓、虎、兔、狗、貓幫你咬敵人
- **🥟 補血包子四散** — 擊殺小兵機率掉落，以拋物線彈飛四散
- **🎵 Web Audio 即時合成音效** — 無需任何音效檔案
- **🏆 本地排行榜** — 通關後輸入名字，依「速度 × 擊殺數 × 剩餘武將」綜合計分，以 localStorage 持久保存
- **📱 手機觸控支援** — 虛擬搖桿 + 觸控按鈕，選角畫面可捲動

## 🚀 如何遊玩

1. 下載 `sanguo_blockworld.html`
2. 用任意瀏覽器開啟（Chrome / Safari / Firefox，需要網路載入 Three.js CDN）
3. 選難度 → 選武將 → 開打！

> **注意：** 遊戲透過 CDN 引入 Three.js r160，開啟時需要網路連線。

## 📐 技術細節

| 項目 | 說明 |
|------|------|
| 引擎 | [Three.js r160](https://threejs.org/) via CDN import map |
| 渲染 | WebGL，PCF 軟陰影 |
| 物理 | 自製簡易地形碰撞 + 動量模擬 |
| 音效 | Web Audio API 即時合成，零外部檔案 |
| 儲存 | localStorage（排行榜持久化） |
| 封裝 | 單一 HTML 檔案，約 4,500 行，無依賴 |
| 光照 | DirectionalLight（可投影）+ AmbientLight，依關卡主題動態切換 |
